# Android Calculator Project

This project contains two versions of the calculator:

1.  **Web App (PWA)**: Located in the root (`index.html`, `script.js`, `style.css`).
    - You can host this or open it in a browser.
    - Installable on Android.

2.  **Android App (Native)**: Located in `android_src/`.
    - This is the source code for the real Android App.
    - You can build the APK using Android Studio or GitHub Actions.

## How to Build APK (Cloud)
1.  Upload ALL files in this folder to GitHub.
2.  Go to "Actions" tab.
3.  Select "Build Android APK".
4.  Download the artifact.
